作品の詳細
■作品1 プラトレ
 ・ゲーム概要
  3Dパズルゲームです。
  イルミネーション(線路)を繋げて、電車を走らせ、人を乗せていき、最後の星を目指すゲームです。
  
■作品2 Sweets Manufacturer
 ・ゲーム概要
  3Dクッキングシミュレーションゲームです。
  ベルトコンベア上の材料をクリックし、ミニゲームを成功させお菓子を完成させていくゲームです。
  
■作品3 栗デレラ
 ・ゲーム概要
  アクションゲームです。
  お正月の前日の夜のこと、
  おせちになることに遅れてしまったイガグリが、次の日までに急いで栗きんとんになるというゲームです。
  
※作品についてvisualstudioが入っていないPCでも確認済みですが、
  VisualStudioでコンパイルして動作させるにはDirectX SDKが必要になります。
